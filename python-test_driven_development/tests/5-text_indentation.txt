Module and Function Description:
================================

The module defines a function to format text indentation.

Function text_indentation(text):
================================


Parameters:
===========

text: A string of text to be formatted.
Behavior:
Checks if text is a string.
Indents the text so that it separates into lines and maintains correct indentation after . , ?, : characters, and newline characters.
