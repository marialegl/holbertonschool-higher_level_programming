4. Module that prints a square with the character #
===================================================

Import the function to be tested
================================
from 4-print_square.py import print_square

Module and Function Description:
================================

The module defines a function to print a square made of hashes (#) of a given size.

Function print_square(size):
============================

Parameters:
===========

size: An integer representing the size of the square.

Behavior:
=========

Checks if size is an integer and greater than or equal to zero.
Prints a square made of size rows and size columns of hashes (#).


Test Cases:
===========

Test case for a valid size:
===========================

size = 5
Expected Output:
#####
#####
#####
#####
#####

Test case for size equal to 0:
==============================

size = 0
Expected Output: (Empty string)

Test case for size less than 0:
===============================

size = -5
Expected to raise a ValueError with the message "size must be >= 0"

Test case for size not being an integer:
========================================

size = "5"
Expected to raise a TypeError with the message "size must be an integer"
